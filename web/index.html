A Premiere Pro panel that watches how you color, learns your style, and grades new footage to match. Powered by Claude Vision, running locally on your Mac.

Download for macOS .dmg · Apple Silicon View on GitHub

What it does

01

Learns your look

Capture frames you've already graded. startup1rit builds a profile of your style — contrast, warmth, how you treat shadows and skin.

02

Grades in context

Point it at a new clip. Claude Vision reads the frame and returns relative Lumetri adjustments that move it toward your style.

03

Stays in budget

A rolling weekly spend cap means it stops calling the API before the bill surprises you. You set the ceiling.

Setup

Download & open the app

Grab the .dmg, drag startup1rit to Applications, launch it. It lives in your menu bar.

Paste your Anthropic API key

Open settings from the menu bar, paste your key, pick a model, set a weekly cap. The local backend starts automatically.

Install the Premiere panel

One click installs the CEP extension. Restart Premiere Pro.

Open the panel

In Premiere: Window › Extensions › startup1rit. Start grading.
